- Seller Login removed from public frontend header/footer links.

- Customer Login / Signup portal.

- Website Install App option.
- Existing PWA install prompt retained.
- Existing design structure retained.
- Seller/admin/payment/GST/FSSAI/reporting features retained.
- Track Order, WhatsApp notifications and cancellation option retained.

Setup:
1. Upload full package files to GitHub.
2. Replace Apps Script with included FreshlyMart_AppsScript.gs.
3. Run setupFreshlyMart().
4. Redeploy Apps Script as a new version.
5. Update assets/app.js BACKEND_URL if needed.
